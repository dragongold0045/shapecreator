Interface & Card Layout

Know where everything lives in the redesigned v8 workspace
01

The editor workspace is divided into five main functional zones:

A
Header Bar (Top)
Contains the shape key input, global Undo/Redo, Theme toggle, Canvas clear, and global data pipeline actions like Import/Export JSON.
B
Context Bar (Below Header)
Provides immediate feedback and fine-tuning controls for your current tool state, grid configurations, subpath start-point drawing toggles, and viewport Zoom rules.
C
Left Panel — Tool Belt & Configuration
Houses drawing modes, geometric presets, the 🤖 AI Trace hub, Smart Snapping coefficients, and the Reference Image matrix setup.
D
Canvas (Center / Interactive Zone)
The high-performance render space supporting advanced node interactions, drag selection boxes, and real-time path calculations.
E
Right Panel — Responsive Card Layout
A flexible layout stack managing Layers & Groups, node context Props, multi-selection coordinate popups, and immediate compilation engine outputs.
🔭

Navigating & View Controls

Pan freely, manage drag selections, and control context parameters
02
1
Zoom & Fluid Pan
Roll the mouse wheel to execute cursor-centric zoom vectors. Hold Middle Click or press H to initiate Pan mode to move across infinite coordinates.
2
Drag Selection Box
Click and drag over an empty area of the canvas to create a bounding selection rectangle, instantly capturing multiple nodes for bulk operations.
3
Multi-Select Floating Menu
When multiple nodes are captured, a smart overlay popup reveals common base controls, enabling you to align, alter, or transform nodes uniformly.

Vector precision, specialized path states, and geometry manipulation
03

Bridges two separate open-ended structural nodes or boundaries seamlessly into a unified compound path.

Smart Snap Engine

Achieve programmatic alignment down to the decimal point
04

The updated layout features an intelligent snapping engine to manage precise canvas coordinates. Toggle quickly via (S).

1
⊕ Axis Alignment Constraints
Pins points safely to the global matrix X=0 and Y=0 grid lines automatically.
2
⊞ Adaptive Grid Snapping
Aligns elements with the active background grid matrix. Set intervals seamlessly via the dropdown panel (0.05, 0.1, 0.25, 0.5).
3
◉ Inter-Layer Node Snapping
Snaps active selections to match vector coordinates across separate layers. Essential for rendering perfect enclosed shapes.

Convert raster images into clean vector layer data instantly
05

Located in the Left Panel, AI Trace uses real-time edge contour-finding algorithms to vectorise transparent image matrices natively.

1

Drop any transparent PNG/WebP element into the canvas interaction portal and trigger the ✨ Trace operation interface.
2

Adjust the alpha threshold cutoff settings to isolate complex textures. Increase the smoothing passes to reduce jagged pixel edges.
3
RDP Simplification Sliders
Leverage the Ramer-Douglas-Peucker value parameters to reduce overall point count while maintaining accurate shape contours.
